This is Choreography,

a library for  view transitions  using native APIs.

Join the Choreography community on Slack

Why?

I believe  Web Apps should be fast,  but also easily understood.

“Properly applied animations between views reduces  cognitive load.”

This is the real war on performance.

For me it is also not just making it  easy to understand for the user,  but also the developer.

But there was no good way  to define and control view transitions.
Especially without giving up control and depending on a framework.

I wanted to do view transitions  in a way that  felt right with the web.
